\# Spun Web Make.com Workflows — Blueprints — Scenarios



This repository provides a curated collection of Make.com scenario blueprints and reusable workflows. Each blueprint is designed to be imported directly into Make.com, helping you automate tasks across marketing, content, scheduling, lead handling, and integrations with popular services.



\## Intent

\- Provide ready-to-import `.json` scenario blueprints for Make.com

\- Accelerate setup of common automation patterns (content generation, social posting, calendars, CRM, notifications)

\- Offer a consistent structure and naming so you can locate and import scenarios quickly



\## Structure

\- Each scenario resides in its own folder and includes a `blueprint.json`

\- Blueprints follow Make.com’s export format and typically contain modules, routes, filters, and mappings



\## Usage (Importing in Make.com)

\- Open Make.com → Scenarios → Create new scenario

\- Click the `•••` menu at the bottom → `Import blueprint`

\- Select the scenario’s `blueprint.json`

\- Connect your integrations and run a test execution
